## About Us

**EventSphere Management** is a leading event and expo management platform designed to revolutionize how expos and trade shows are organized, experienced, and remembered.

We connect **organizers**, **exhibitors**, and **attendees** in a seamless digital ecosystem — eliminating manual processes, disjointed communication, and limited real-time access to event information.

With over **15,000+ successful events** managed, our platform empowers you to create memorable experiences that inspire, connect, and deliver results.

## Getting Started

### Prerequisites
- Node.js v18+
- MongoDB installed locally or cloud URI
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Install backend dependencies
cd backend
npm install

# Configure environment variables
cp .env.example .env
# Edit .env with your MongoDB URI and JWT secret

# Start the backend server
npm start

# In a new terminal, install frontend dependencies
cd ../frontend/Website_Ui_Design
npm install

# Start the frontend
npm start
```

The application will be available at `http://localhost:3000`
